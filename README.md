# متجر تطبيقات وسيل بلاي

## وصف المشروع
وسيل بلاي - متجر التطبيقات العربي المجاني لتحميل أفضل التطبيقات والألعاب للاندرويد. 
منصة عربية متكاملة توفر أحدث التطبيقات في جميع الفئات مجاناً.

## الميزات
- ✅ عرض التطبيقات والألعاب
- ✅ بحث وتصفية متقدمة
- ✅ لوحة تحكم للمسؤول
- ✅ نظام مشاركة التطبيقات
- ✅ تحميل مباشر وسريع
- ✅ دعم كامل للغة العربية
- ✅ تحسين محركات البحث (SEO)

## التقنيات المستخدمة
- Firebase Firestore
- Firebase Authentication  
- Firebase Storage
- JavaScript (ES6+)
- HTML5 / CSS3
- Responsive Design

## كيفية التشغيل
1. افتح `index.html` في المتصفح
2. أو انتقل إلى: https://wacel-play.netlify.app
3. البدء في تصفح وتحميل التطبيقات

## فئات التطبيقات
- الألعاب
- التواصل الاجتماعي
- الترفيه
- فوتوغرافي
- الذكاء الاصطناعي
- الأدوات المساعدة
